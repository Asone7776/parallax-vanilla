Performance is key

Maximize your parallax effects with hardware acceleration and no external libraries.

pv-height = "40vh" | pv-speed = [-1.0, -1.1, .., -2.1] (12 elements)
pv-height = "10vh" | pv-speed = [1.0, 1.1, .., 3.3] (24 elements)
pv-height = "10vh" | pv-speed = [-1.0, -1.1, .., -3.3] (24 elements)
pv-height = "40vh" | pv-speed = [1.0, 1.1, .., 2.1] (12 elements)

Viewport-only animations

Parallax elements are only animated within the current viewport, saving enormous resources.

pv-height = "400" | pv-speed = -1

Dynamic sizing

Elements are dynamically sized and adjusted relative to the pv-speed.
